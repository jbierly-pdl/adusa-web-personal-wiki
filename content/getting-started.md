# Getting Started

Welcome to your personal wiki! This is a sample note to verify that Nuxt Content is working correctly.

## How It Works

1. **Drop markdown files** into the `content/` directory
2. **Add frontmatter** for metadata like tags, dates, and categories
3. **Browse your wiki** — every file becomes a page automatically

## What You Can Do

- Write notes in standard Markdown
- Organize with tags and categories
- Link between notes using standard markdown links
- Use MDC syntax to embed Vue components inside your content

## Next Steps

Try creating a new `.md` file in the `content/` directory and see it appear in your wiki!
